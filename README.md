# Création d'un boutton dynamique


 
## Comment utiliser mon application:

Ecrire dans la barre de recherche le nom de pokemon voulu (en anglais): une image, le nom et le type s'affichera.

## Technologies:

* HTML
* CSS
* Javascript
* API: [PokeAPI](https://pokeapi.co)
